📋 Sensor Setup Guide

📡 Soil Moisture Sensor

मिट्टी की नमी माप — FC-28 या Capacitive sensor खेत में 15-20 सेमी गहराई पर लगाएँ। किसान AI app से connect करें।

🌡️ Temperature Sensor (DHT22)

मिट्टी और हवा का तापमान। DHT22 sensor छाँव में लगाएँ। Data हर 15 minute में update होता है।

🧪 pH Sensor

मिट्टी की अम्लता जाँचें। Ideal range 6.0-7.5। इससे कम हो तो चूना, ज़्यादा हो तो गंधक मिलाएँ।

🔧 Setup कैसे करें

Arduino / ESP32 से sensors connect करें → किसान AI app mein IoT section खोलें → Device pair करें → Real-time data देखें।
